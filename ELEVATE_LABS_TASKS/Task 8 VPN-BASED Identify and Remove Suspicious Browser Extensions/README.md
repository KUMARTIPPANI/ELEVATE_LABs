# 🌐 Cyber Security Internship — Task 8  
## Use of VPN for Privacy and Secure Communication  

### 🎯 Objective  
The goal of this task was to understand how a Virtual Private Network (VPN) works, how it helps protect online privacy, and to test its effect on IP address and browsing behavior.

### 🧰 Tools Used  
- **VPN Used:** ProtonVPN (Free Version)  
- **Browser:** Google Chrome  
- **Device:** Windows Laptop  
- **Website for IP Check:** [whatismyipaddress.com](https://whatismyipaddress.com/)  
- **Website for Testing:** [wikipedia.org](https://www.wikipedia.org/)  

### 🧭 Steps I Followed  
1. Downloaded and installed **ProtonVPN Free Version** from its official website.  
2. Logged in and connected to different servers — **United States (America)** and **Japan** — to test VPN performance.  
3. Opened **[whatismyipaddress.com](https://whatismyipaddress.com/)** to check my public IP address before turning on the VPN.  
4. After connecting to the VPN, I checked the IP again and noticed the change in both **IP location** and **ISP details**, confirming the VPN connection.  
5. Visited **[wikipedia.org](https://www.wikipedia.org/)** with VPN turned ON to ensure secure browsing and encrypted traffic.  
6. Disconnected the VPN and checked the IP once more — it reverted to my actual IP, showing that ProtonVPN effectively masked my real location.  
7. Repeated the same process for the Japan server to verify multiple connections.  
8. Observed a slight reduction in browsing speed, which is normal due to encrypted tunneling.

### 🔍 Findings  
- The VPN successfully changed my IP address and masked my real location.  
- My traffic appeared to be encrypted, and websites detected me as browsing from another country.  
- Browsing speed slightly decreased while connected to distant servers (like Japan).  
- ProtonVPN provided stable and secure connections with no data leaks during the test.  

| **VPN Server** | **Detected Location** | **Result** |
|-----------------|-----------------------|-------------|
| America         | U.S. IP detected      | Successful |
| Japan           | Japan IP detected     | Successful |
| VPN Off         | My real IP visible    | Expected behavior |

### 🧠 What I Learned  
This task helped me understand how VPNs protect user privacy by encrypting data and hiding the real IP address. I learned how VPNs create a secure tunnel between the user and the internet, ensuring data confidentiality and anonymity. I also realized that while VPNs improve privacy, they can slightly slow down network speed depending on the server location.

### ✅ Conclusion  
By completing this task, I gained hands-on experience using ProtonVPN and verified its effectiveness in securing internet connections and hiding IP addresses. The experiment clearly showed the difference between connected and disconnected VPN states. This task improved my understanding of **VPN encryption, privacy, and secure browsing**.
