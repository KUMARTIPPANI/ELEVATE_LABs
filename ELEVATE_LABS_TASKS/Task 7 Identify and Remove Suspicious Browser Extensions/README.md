# 🧩 Cyber Security Internship — Task 7  
## Identify and Remove Suspicious Browser Extensions  

### 🎯 Objective  
The goal of this task was to identify and analyze the installed browser extensions, check for any suspicious or unnecessary ones, and understand how browser extensions can pose security risks. This task helped me learn about browser privacy, extension permissions, and how to ensure safe browsing.

### 🧰 Tools Used  
- **Browser:** Google Chrome  
- **System:** Windows  
- **Tool Used:** Chrome Extensions Manager (`chrome://extensions/`)  

### 🧭 Steps I Followed  
1. I opened Google Chrome and typed **chrome://extensions/** in the address bar to access the extensions manager.  
2. I carefully reviewed all installed extensions visible in the browser.  
3. I clicked on the **Details** option for each extension to check its permissions, origin, and purpose.  
4. I verified whether each extension was downloaded from a trusted source and reviewed their purpose.  
5. I researched online about each extension to ensure it was safe and not known for any malicious activity.  
6. I identified three extensions installed on my browser:  
   - **Adblock Plus — Free Ad Blocker**  
   - **Adobe Acrobat — PDF Edit, Convert, Sign Tools**  
   - **Grammarly — AI Writing Assistant**  
   - **Google Docs Offline**  
   - **PDF Viewer Pro**  
7. After checking their permissions and authenticity, I found that all these extensions were **safe, verified, and widely used**. None of them showed suspicious permissions or risky behavior.  
8. I ensured that no unknown or untrusted extensions were installed, and my browser was free from potential threats.  

### 🔍 Findings  
All extensions were legitimate and safe to use. None required unusual permissions such as file system access, camera/microphone control, or remote script execution. Extensions like Grammarly and Adobe Acrobat were verified publishers on the Chrome Web Store, and Adblock Plus had a large user base with positive reviews. Therefore, no extensions needed to be removed.

| -------------------- | ---------- | ------------ |
| Adblock Plus | Safe | Verified and trusted extension |
| Adobe Acrobat | Safe | Official Adobe extension for PDF tools |
| Grammarly | Safe | Verified writing and grammar tool |
| Google Docs Offline | Safe | Google official productivity extension |
| PDF Viewer Pro | Safe | Allows PDF viewing and editing securely |

### 🧠 What I Learned  
From this task, I learned how browser extensions can access user data and sometimes exploit permissions for malicious activities such as stealing personal data or injecting ads. I also understood how to check an extension’s permissions, developer information, and reviews to verify its safety. It’s important to only install extensions from official stores and disable or remove those not in use.  

This task improved my awareness of **browser security**, **extension permissions**, and **data privacy**. I also learned that malicious extensions can perform unauthorized actions like tracking browsing history or capturing keystrokes if permissions are misused.

### ✅ Conclusion  
By completing this task, I gained hands-on experience in identifying, reviewing, and managing browser extensions securely. All extensions on my Chrome browser were confirmed safe and functional. This activity enhanced my understanding of how extensions can impact browser performance and privacy and reinforced the importance of regular security checks.
