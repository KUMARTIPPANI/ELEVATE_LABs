# 🔐 Cyber Security Internship — Task 6  
## Create a Strong Password and Evaluate Its Strength  

### 🎯 Objective  
The objective of this task was to understand what makes a password strong and to test different passwords using an online password strength checker. This activity helped me learn how factors such as password length, use of uppercase and lowercase letters, numbers, and special symbols impact overall security and resistance to attacks.

### 🧰 Tools Used  
- Website: [https://passwordmeter.com/](https://passwordmeter.com/)  
- Platform: Web Browser (Google Chrome)  
- System: Windows  

### 🧭 Steps I Followed  
I started by opening the website **https://passwordmeter.com/**, a free online password strength testing tool. The site provides an instant score and feedback based on how strong or weak a password is. I created multiple passwords with varying levels of complexity and tested each one on the website to analyze the difference in their strength scores and feedback.  

Here are the passwords I tested and their results:  

| **Password Example** | **Description** |
| -------------------- | ---------------------------------------------- |
| eleveatelabs123 | Weak — simple, short |
| Eleveate@123 | Medium — uses uppercase and symbol |
| ElEvE@T#2025 | Strong — mixed uppercase, symbols, and numbers |
| MyP@ssw0rd!2025$# | Very Strong — long and complex |

After entering each password, the website displayed a percentage score along with suggestions on how to improve strength, such as adding more characters, using mixed case letters, and including symbols or numbers. I observed that longer passwords with a combination of uppercase, lowercase, symbols, and numbers received higher strength ratings.

### 🧠 What I Learned  
Through this task, I learned how password complexity plays a major role in protecting accounts from hacking attempts. Simple passwords can be easily broken using **brute force** or **dictionary attacks**, whereas complex and longer passwords are much more secure. I also learned about **best practices** for password creation — using at least 12–16 characters, avoiding personal information, and including multiple types of characters.  

Additionally, I explored the concept of **multi-factor authentication (MFA)**, which adds another layer of protection beyond the password. I also learned about **password managers**, which securely store strong, unique passwords for each account. The task also introduced me to **passphrases**, which are longer combinations of random words that are both strong and easy to remember.

### ✅ Conclusion  
This task helped me understand the science behind strong password creation and how it directly affects the safety of online accounts. By testing different passwords on **passwordmeter.com**, I clearly saw how complexity improves resistance to attacks. I realized that using a mix of uppercase, lowercase, numbers, and symbols — along with good password management habits — can significantly improve security. This task strengthened my awareness of authentication best practices and password safety in cybersecurity.
